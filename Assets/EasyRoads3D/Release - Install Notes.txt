IMPORTANT:

Install Notes:

To install this update in an existing EasyRoads3D project it is recommended to follow the next steps:

1. Backup your project
2. Open a new scene if there is an EasyRoads3D Road Network object present in the currently open scene
3. Simply import package updates over the existing install without first removing the previous package
4. Open your scene
